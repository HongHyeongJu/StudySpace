비동기식 자바스크립트 XML
브라우저측 자바스크립트
에이잭스 기능을 활용하는데 도움이 되는 2가지 내장 브라우저 측 자바스크립트
1. XML_HTTP_Request
2. fetch()

자바스크립트 코드 내부에서 서버로 HTTP요청을 보낼 수 있음


axios는 자바스크립트에서 사용되는 인기 있는 HTTP 클라이언트 라이브러리
웹 서버와 비동기적으로 데이터를 주고받기 위한 Ajax 요청을 손쉽게 할 수 있도록 도와줌!

Axios의 주요 특징
Promise 기반: Axios는 JavaScript의 Promise API를 사용합니다. 이를 통해 비동기 코드를 보다 쉽게 작성하고 관리할 수 있습니다.
브라우저와 Node.js에서 모두 사용 가능: Axios는 브라우저뿐만 아니라 Node.js에서도 사용할 수 있어, 서버 사이드와 클라이언트 사이드 모두에서 활용 가능합니다.
자동 JSON 데이터 변환: 서버로부터 받은 데이터를 자동으로 JSON 형태로 변환해줍니다. 또한, 서버에 데이터를 보낼 때 JSON 형태로 자동 변환해 보낼 수 있습니다.
요청 취소, HTTP 요청의 타임아웃 설정: 요청을 취소하거나, 요청이 너무 오래 걸릴 때 타임아웃을 설정할 수 있습니다.
인터셉터: 요청과 응답을 가로채어 추가적인 로직을 수행할 수 있는 기능을 제공합니다. 예를 들어, 모든 요청에 특정 HTTP 헤더를 추가하거나, 응답을 받았을 때 특정 작업을 수행하는 것이 가능합니다.
와이드 브라우저 지원: 다양한 웹 브라우저에서 사용될 수 있도록 설계되었습니다.
